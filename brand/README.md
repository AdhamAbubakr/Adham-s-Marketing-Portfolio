# Barkar. — Brand Assets

Source files for the **Barkar.** marketing agency brand identity.
Founded by **Adham Abo Bakr SalahElden**.

> **Barkar** is a coined brand name &mdash; built to stand out, defined by the work, not by dictionary meaning.
>
> Tagline: **"Built different."**

## 📁 Files

| File | Use case |
|---|---|
| `logo-primary.svg` | Horizontal logo (mark + wordmark + tagline) |
| `logo-stacked.svg` | Vertical / centered logo |
| `logo-mark.svg` | Symbol only — small avatars (64px+) |
| `logo-white.svg` | Mono white version — dark backgrounds |
| `logo-black.svg` | Mono black version — light backgrounds |
| `favicon.svg` | Tiny version for browser tab (32px) |
| `social-avatar.svg` | 1000×1000 ready for IG / X / LinkedIn |

## 🔤 Typography

- **Display:** Space Grotesk (700) — wordmark, headlines, big numbers
- **Body:** Inter (400 / 500 / 600) — paragraphs, captions, UI

Open `brand-guide.html` in a browser → "Save as PDF" to export the full brand book.

## ⚖️ Quick rules

- The "**A**" mark works for both the wordmark "Barkar" and the founder "Adham"
- Always include the magenta dot in the wordmark (Barkar**.**)
- Don't recolor outside the palette
- Don't add drop-shadows or 3D effects
- Minimum size: 24px (use the favicon below that)
