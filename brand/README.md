# Adham. — Brand Assets

Source files for the **Adham.** brand identity. Use these directly in any design — don't recreate them.

## 📁 Files

| File | Use case |
|---|---|
| `logo-primary.svg` | Horizontal logo (mark + wordmark + tagline) — websites, decks, email signatures |
| `logo-stacked.svg` | Vertical / centered logo — covers, posters, square placements |
| `logo-mark.svg` | Symbol only — small avatars, app icons (64px+) |
| `logo-white.svg` | Mono white version — for dark photo backgrounds |
| `logo-black.svg` | Mono black version — for light backgrounds |
| `favicon.svg` | Tiny version for browser tab (32px) |
| `social-avatar.svg` | 1000×1000 ready-to-export profile pic for IG / X / LinkedIn |

```
Royal Purple      #7C3AED   (Primary)
Electric Violet   #A855F7   (Highlight)
Magenta Spark     #EC4899   (Accent / energy)
Cyan Pulse        #06B6D4   (Data)
Cream             #FEF3C7   (Warm contrast)
Midnight          #0A0118   (Background)
```

## 🔤 Typography

- **Display:** Space Grotesk (700) — headlines, the wordmark, big numbers
- **Body:** Inter (400 / 500 / 600) — paragraphs, captions, UI

Both fonts are free via Google Fonts.

## 📖 Full guide

Open `brand-guide.html` in any browser → click "Save as PDF" (top-right) to export the full brand-book as a PDF.

## ⚖️ Quick rules

- Never recolor the mark with off-brand colors
- Always include the magenta period in the wordmark (Adham**.**)
- Don't add drop-shadows, 3D, or rotate the mark
- Minimum size: 24px (use favicon below that)
